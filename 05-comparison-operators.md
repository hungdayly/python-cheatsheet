Các toán tử so sánh được dùng để so sánh các giá trị. Kết quả trả về của chúng là `True` hoặc `False`.

```python
# So sánh bằng
number = 5
assert number == 5

# So sánh không bằng
number = 5
assert number != 3

# Lớn hơn
number = 5
assert number > 3

# Nhỏ hơn
number = 5
assert number < 8

# Lớn hơn hoặc bằng
number = 5
assert number >= 5
assert number >= 4

# Nhỏ hơn hoặc bằng
number = 5
assert number <= 5
assert number <= 6
```
